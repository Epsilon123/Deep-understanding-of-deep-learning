# 卷积
卷积的数学表达是一个加权求和的过程，利用卷积可以提取图片中颜色，纹理，形状和位置（上下文）等特征，从而进行分类。带有卷积操作的神经网络就是CNN，通过卷积提取图像中的特征，使用ANN（全连接层）进行分类。从数学表达上看，ANN是一个一维的加权求和的过程，在使用ANN进行字符识别的任务中，输入的字符图片被拉伸为一维向量，输入到ANN中，这种方法参数多，特征提取能力弱，难以实际使用。利用卷积提取特征，实现维度的极大降低，这种方法参数少，特征提取能力强，使CNN在ILSVRC比赛中大放异彩，但实用性还有待提高。

# 转置卷积
卷积和ANN的数学表达都是加权求和操作，所以可以使用卷积替换CNN中的全连接层，使用局部连接的卷积替换全连接层，可以显著降低参数量，这就是全卷积神经网络（FCN），它的输出是二维的图像，可以实现对每一个像素进行分类。FCN的实用性比CNN强很多，是真正可以实际应用的神经网路。在unet中，整个结构被看成是一个编码器-解码器，左边的卷积是编码模块，右边转置卷积是解码模块，每个卷积/转置卷积都一一对应。卷积的数学表达是加权求和，也可以用矩阵乘表示，转置卷积是卷积的反向操作，可以看成是求矩阵的逆，所以有人称转置卷积为反卷积或者逆卷积，这是不准确的。“转置卷积是卷积的反向操作”是一种思想，把输入数据和label看成是相同的东西（通过映射相互转换），但实际上两者不一样。这种思想的确实现了卷积的可视化和密集预测，“转置卷积是卷积的反向操作，可以看成是求矩阵的逆”，矩阵的逆也是矩阵，矩阵逆的乘法也是矩阵的乘法，也是卷积操作，所以，转置卷积就是卷积，转置卷积是一个概念上的东西。所以说，FCN可以只包含卷积操作，卷积既可以提取特征，也可以实现分类。
